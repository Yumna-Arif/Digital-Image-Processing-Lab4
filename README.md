# 🖼️ Image Steganography using Python (DIP Lab)

## 📌 Overview
This project demonstrates **LSB-based Image Steganography** using Python and OpenCV.  
It also includes concepts of **Pixel Neighbors** and **Connectivity** from Digital Image Processing.

---

## 🚀 Features
- Hide one image inside another using bit manipulation
- Extract hidden image
- Understand 4-neighbors and 8-neighbors
- Connectivity check between pixels

---

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib

---

## 📷 Output
- Stego Image
- Extracted Hidden Image

---

## 🎯 Concepts Covered
- LSB (Least Significant Bit)
- Bit Shifting
- Bitwise Operations
- Pixel Connectivity

---

## 📚 Applications
- Secure Communication
- Digital Watermarking
- Image Processing
